# Classification-using-Machine-Learning---Deep-Learning
# Kitchen Objects Classification using Machine Learning & Deep Learning

## 📌 Overview

This project was developed as part of the **AI3011 – Artificial Intelligence and Machine Learning** course.

The objective is to classify kitchen objects into three categories:

- 🥤 Cup
- 🍽️ Plate
- 🥄 Spoon

The project compares Classical Machine Learning algorithms with Deep Learning models and analyzes their performance using various evaluation metrics.

---

## 🎯 Objectives

- Explore and analyze the dataset.
- Preprocess image data.
- Extract handcrafted image features.
- Train multiple Machine Learning models.
- Train a Deep Learning model.
- Compare model performance.
- Explain model predictions (XAI).
- Deploy the best-performing model for inference.

---

## 📂 Dataset

The dataset contains images of three kitchen object classes:

- Cup
- Plate
- Spoon

The images are divided into:

- Training Set
- Validation Set
- Test Set
